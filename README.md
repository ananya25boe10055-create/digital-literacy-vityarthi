Project Overview
This project documents five practical tasks completed as part of the Digital Literacy and Online Safety course (CSE0001) at VIT Bhopal University. The project equips first-year students with the knowledge and skills to navigate the digital world safely, responsibly, and effectively.
The tasks collectively cover digital content creation, professional online identity, technical skill building, formal digital communication, and cybercrime awareness — using tools such as Canva, LinkedIn, GitHub, Kaggle, HackerRank, and Google Forms.

Module Summaries

Task 1 — Digital Literacy Presentation
Tool Used: Canva
An 8-slide presentation on Digital Literacy and Safe Internet Practices, covering the five key components of digital literacy (technical skills, information literacy, communication skills, digital ethics, and cyber awareness), recommended academic tools, and safe internet practices including password hygiene and digital footprints.

Task 2 — Professional Online Presence
Platforms: LinkedIn · Kaggle · GitHub
Set up and documented three professional profiles to build a credible digital identity:
LinkedIn — Academic background, skills, and professional networking
Kaggle — Data science portfolio and ML competition participation
GitHub — Code repository for biomedical and engineering projects

A long-term 4-year plan outlines progressive goals on each platform, culminating in an active biomedical ML project portfolio.

Task 3 — Digital Tools for Academic Use
Platforms: Google Forms · HackerRank

Built a Digital Literacy Awareness Quiz on Google Forms (3 questions, 30 points total) covering digital literacy definitions, safe internet practices, and phishing awareness.
Earned the Python (Basic) Certification on HackerRank (profile: @ananya_25boe1001), demonstrating foundational programming competency relevant to the Bioengineering curriculum.

Task 4 — Digital Communication Skills
Covered two aspects of professional digital communication:
Task 4A — Formal Email Writing
Drafted two professional emails: (1) a deadline extension request to a professor for a Human Physiology assignment, and (2) an internship interest email to an organisation — both following structured, respectful formatting with clear subject lines, salutations, and closings.
Task 4B — Responsible Social Media Use
Compiled a Do's and Don'ts checklist for college students, emphasising accuracy, tone, privacy, and professionalism in all online interactions.

Task 5 — Cybercrime Awareness & Online Safety
Task 5A — UPI Payment Fraud Case Study
Analysed a real-world OLX scam where a victim lost ₹45,000 via a fake UPI "collect request." Key insight: you never need to enter your UPI PIN to receive money — a fundamental rule that prevents this class of fraud entirely.
Task 5B — Stay Safe Online Checklist
A practical safety checklist for college students in India, including guidance on verifying payment requests, spotting phishing, and reporting cybercrime through official channels.
